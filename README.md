## ⚡ Current Work

신재생 에너지 회사에서 아래 서비스들을 개발 및 운영하고 있습니다.

- **클라이언트 맞춤형 API** — 다양한 요구사항을 분석하여 확장 가능한 RESTful API 설계 및 구현
- **신재생 에너지 설비 제어 서비스** — EMQX 기반 MQTT 프로토콜을 활용한 실시간 설비 제어 시스템
- **데이터 취득 서비스** — 현장 센서 데이터의 실시간 수집 및 저장 파이프라인 구축
- **기상 수집 서비스** — 외부 기상 API 연동 및 주기적 데이터 수집·가공 시스템
- **신재생에너지 입찰 서비스** — 전력 거래 입찰 프로세스 자동화 및 서비스 로직 구현
- **신재생 유연시장 서비스** — 유연성 자원 거래를 위한 시장 연동 및 데이터 처리 시스템
- **단방향 데이터 송수신 서비스** — 폐쇄망 환경에 단방향 장치를 설치하여 신재생 에너지 데이터를 안전하게 수집하고 제어 데이터를 전송하는 보안 특화 서비스
